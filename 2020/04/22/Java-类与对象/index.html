类的定义和使用

预定义的类

对于时间，标准 Java 类库包含两个类：Date 类表示时间，LocalDate 类表示日期。

LocalDate 类的 API：

  • LocalDate.now()：当前日期，如 2020-04-22

  • LocalDate.of(int year, int month, int day)：指定日期

  • int getYear()：年

  • int getMonthValue()：月

  • int getDayOfMonth()：日

  • DayOfWeek getDayOfWeek()：星期几

    • getValue()：1～7，表示星期一～星期日

  • LocalDate plusDays(int n)：加 n 天

  • LocalDate minusDays(int n)：减 n 天

自定义类

一个源文件中，只能有一个公有类，但是可以有无限个非公有类。

一个方法中，隐式参数用this表示，就是在方法前面（一般在类的开头那）已经定义好的对象。

静态属性和方法

静态就是static，它属于类且不属于类对象的变量和函数。

若将某个字段定义为static静态域，则这个类所属的对象实例们将共享这一个静态域。即静态域属于类，不属于对象。

静态方法操作的是类，它不能操作对象。若某对象调用静态方法，那么计算的结果与这个对象无关。

Java 中 4 个访问修饰符的范围：

  • public：所有类
  • protected：本包和所有子类可见
    • 适用不提供一般途径，而应在子类中重新定义的方法
  • 默认：本包可见
  • private：仅本类可见

泛型数组

泛型数组不需要制定数组的长度，若数组空间满了，则会自动创建一个更大的数组。

若已经确定了元素的数量，可以在填充元素之前就设置好ArrayList的大小，可以大大提高效率：

若确定数组列表大小不再变化，可以调用trimToSize()来锁定当前数组大小，这样可以通过GC回收多余的存储空间。

ArrayList 的常用函数：

  • void set(int index, E obj)：覆盖数组中已经存在的元素值
  • E get(int index)：得到指定位置的元素值
  • void add(int index, E obj)：添加值，向后移动元素（不适合大数值，大数值用链表）
  • E remove(int index)：删除值，后面的元素向前移动（被删除的元素又返回值返回）

对于枚举类的比较，使用==，而不要使用equals，因为每个实例在 JVM 都是唯一的。
