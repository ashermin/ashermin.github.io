基本变量类型

Java 是强类型语言，必须要给每个变量声明一种类型。

一共有 8 种基本类型：4 种整型、2 种浮点型、1 种字符型和 1 种布尔类型。

1. 整型

类型 存储需求 取值范围
int 4 字节 -2147483648 ~ 2147483647
short 2 字节 -32768 ~ 32767
long 8 字节 -9223372036854775808 ~ 9223372036854775807（结尾需要加L/l）
byte 1 字节 -128 ~ 127

当 long 都满足不了时，可以使用 BigInteger 类。

2. 浮点型

类型 存储需求 取值范围 有效位数
float 4 字节 大约 ± 3.402 823 47E+38F （结尾需要加F/f） 6～7 位
double 8 字节 大约 ± 1.797 693 134 862 315 70E+308 15 位

⚠️注意：float 和 double 只能用来做科学计算或者是工程计算；在商业计算中要用java.math.BigDecimal来解决

2.5 BigInteger 和 BigDecimal 类

3. 字符型 char

char 表示 Unicode 编码的单个字符，是用单引号的，如'A'（编码值为65所对应的字符常量）；而"A"是包含一个字符 A 的字符串。

⚠️注意：不建议使用 char 类型！

4. 布尔型

只能是 false 和 true，不能用整型值来代替。

字符串 String

1. String API

String 类的所有方法可以参见 Oracle 官网。

（1）返回字符串中指定位置的字符

  • char charAt(int index)

（2）两字符串按字典序比较

  • int compareTo(String other)
    • 字符串位于 other 之前，返回负数
    • 字符串位于 other 之后，返回正数
    • 相等，返回 0

（3）判断字符串内容是否相等

  • boolean equals(Object other)
  • 忽略大小写：boolean equalsIgnoreCase(String other)

⚠️注意：

  1. 不要使用==检测两个字符串是否相等，它判断的是对象地址是否相同；
  2. 如果a是null，则会报错。可以利用&&来解决或者把非null对象放前面：

（4）判断字符串是否以 xxx 开头/结尾

  • boolean startsWith(String str)
  • boolean endsWith(String str)

（5）从左往右查询与字符串匹配的第一个子串的开始位置

  • 从索引 0 开始：int indexOf(String str)
  • 从索引 fromIndex 开始：int indexOf(String str, int fromIndex)
  • 不存在子串，返回 -1

（6）从右往左查询与字符串匹配的最后一个子串的开始位置

  • 从最右开始往左：int lastIndexOf(String str)
  • 从指定的 fromIndex 开始往左：int lastIndexOf(String str, int fromIndex)

虽然指定的第 3 个是a，但是它后面也是完整的ab，所以也算！

（7）返回字符串的长度

  • int length()

（8）字符串的替换

  • String replace(CharSequence old, CharSequence new)

（9）字符串的截取

  • 从 beginlndex 到最后：String substring(int beginIndex)
  • 从 beginlndex 到 endlndex-1：String substring(int beginIndex, int endIndex)

（10）字符串的大小写

  • 全小写：String toLowerCase()
  • 全大写：String toUpperCase()

（11）删除字符串的空格

  • String trim()

2. StringBuilder 类

许多小段的字符串拼接，使用普通的字符串拼接每次都会构建一个新的 String 对象，既耗时又占空间。可以采用 StringBuilder 类，最后调用toString()方法得到一个 String 对象。

下面是 StringBuilder 类的常用 API：

（1）返回字符数

  • int length()

（2）将第 i 个字符设为 c

  • void setCharAt(int i, char c)

（3）指定位置插入一个字符串/字符

  • StringBuilder insert(int offset, String str)
  • StringBuilder insert(int offset, Char c)

（4）删除指定范围的字符

  • StringBuilder delete(int startIndex, int endIndex)
    • 范围：[startIndex, endIndex)

3.1 控制台输入

构造一个 Scanner 对象，并与“标准输入流” System.in 关联。

Scanner 对象常用的方法：

  • 输入的下一行：String nextline()
  • 输入的下一个单词（以空格为分隔符）：String next()

  • 检测输入中是否还有其它数字：boolean hasNextInt()
  • 检测输入中是否还有其它浮点数：boolean hasNextDouble()

⚠️注意：当使用的类不是在基本 java.lang 包中时，就要引入。
