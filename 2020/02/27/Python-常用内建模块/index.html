1. datetime

datetime是Python处理日期和时间的标准库。

1.1 获取当前/指定日期和时间

  • datetime.now()返回当前日期和时间，其类型是datetime
  • 要指定某个日期和时间，直接用参数构造一个datetime

1.2 datetime与timestamp的转换

  • datetime转换为timestamp：调用timestamp()方法
  • timestamp转换为datetime：
    • 本地时区：调用datetime.fromtimestamp()方法
    • UTC标准时区：调用datetime.utcfromtimestamp()方法

1.3 datetime与str的转换

  • datetime转换为str：调用strftime()方法 + 时间/日期字符串
  • str转换为datetime：调用datetime.strptime()方法 + 时间/日期字符串

字符串部分的代码格式可以参考官网：https://docs.python.org/3/library/datetime.html#strftime-strptime-behavior

2. collections

collections是Python内建的一个集合模块，提供了许多有用的集合类。

2.1 namedtuple

namedtuple是一个函数，它用来创建一个自定义的tuple对象，并且规定了tuple元素的个数，并可以用属性而不是索引来引用tuple的某个元素。

语法格式：namedtuple('名称', [属性list])

2.2 deque

deque和list都是队列，不过deque是双向队列，而list是单向；在删除和插入时，deque很高效，但由于list是线性存储的，数据量一大时效率就很低。

只要list里有的方法，deque都可以用，而且deque还有自己的方法：

  • 头部添加元素：appendleft(x)
  • 头部添加多个元素：extendleft(iterable)
  • 头部返回并删除：popleft()
  • 旋转：rotate(n=1)
  • 最大空间：maxlen，如果无边界返回None
