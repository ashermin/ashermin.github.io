1. Kafka基本概念

1.1 Kafka中的基本术语

Producer

消息生产者，往某个Topic上发布消息，也负责选择发布到Topic的哪一个分区。

Consumer

消息消费者，一个发布在Topic上的消息允许广播给多个消费者组，但是每个消费者组只能有一个消费者可消费该消息。

Broker

Kafka集群中的每个Kafka节点服务器。每一个Broker保存Topic的一个或多个Partitions

IO流是一种流式的数据输入/输出模型：

  • 二进制数据以byte为最小单位在InputStream/OutputStream中单向流动

File对象

Java集合

发表于 2019-10-27 分类于 Java

Java集合

List

有序链表。即：长度可变的数组，允许重复

实现List接口的有ArrayList和LinkedList，一般优先用ArrayList

List的主要接口方法：

  • 在末尾添加一个元素：void add(E e)
  • 在指定索引添加一个元素：void add(int index, E e)
  • 删除指定索引的元素：int remove(int index)
  • 删除某个元素：int remove(Object e)
  • 获取指定索引的元素：E get(int index)
  • 获取链表大小（包含元素的个数）：int size()
