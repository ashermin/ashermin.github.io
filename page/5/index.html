Flume基础

发表于 2019-11-26 更新于 2020-01-01 分类于 Flume

Flume是分布式日志收集框架，用于分布式的海量日志的高效收集、聚合和移动系统。

这里我使用的版本是flume-ng-1.6.0-cdh5.7.6 官方用户手册

1. Flume架构及核心组件

Flume由三大核心组件构成：Source、Channel、Sink

  • Source：收集。指定数据的来源，比如：Web Server
  • Channel：聚集。为数据提供临时存放的缓存通道，Source来的数据存到Channel满了以后再发送到Sink
  • Sink：输出。把数据写到某个地方去，比如：HDFS

Kafka基本使用

发表于 2019-11-21 更新于 2020-01-01 分类于 Kafka

1. Kafka基本概念

1.1 Kafka中的基本术语

Producer

消息生产者，往某个Topic上发布消息，也负责选择发布到Topic的哪一个分区。

Consumer

消息消费者，一个发布在Topic上的消息允许广播给多个消费者组，但是每个消费者组只能有一个消费者可消费该消息。

Broker

Kafka集群中的每个Kafka节点服务器。每一个Broker保存Topic的一个或多个Partitions
