二、基本数据类型

计算机内存的最小存储单元是字节（byte），一个字节就是8位二进制数（bit），即从00000000至11111111或0至255。（注：1024byte=1K，1024K=1M）

Java定义了以下几种基本数据类型（所占的字节数用括号表示）：

  • 整数类型：byte(1)，short(2)，int(4)，long(8)

  • 浮点数类型：float(4)，double(8)

  • 字符类型：char(2)

  • 布尔类型：boolean

1. 整型

由于符号，所以最高位的bit为符号位（0+，1-）。各种整型的范围如下：

  • byte：-128 ~ 127
  • short: -32768 ~ 32767
  • int: -2147483648 ~ 2147483647
  • long: -9223372036854775808 ~ 9223372036854775807（结尾需要加L）

注意：Java中求幂不能用a^b（a与b异或运算，同位取0不同取1），应该是Math.pow(a,b)。

2. 浮点型

  • float：32位，4字节（结尾需要添加f）
  • double：64位，8字节
3. 布尔类型 boolean
  • true
  • false
4. 字符类型 char

用单引号表示，并且只能保存一个字符。

  • char：保存一个字符
5. 常量 final

初始化后不可再赋值，通常全部大写。

*2. 数组排序 *

Arrays.sort() 可以实现对数组的排序，默认为升序。

（1）Arrays.sort(an)（默认升序）

（2）Arrays.sort(an,fromIndex,toIndex) 在区间[ fromIndex , toIndex )的范围内对数组进行升序排序
