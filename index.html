类的定义和使用

预定义的类

对于时间，标准 Java 类库包含两个类：Date 类表示时间，LocalDate 类表示日期。

LocalDate 类的 API：

  • LocalDate.now()：当前日期，如 2020-04-22

  • LocalDate.of(int year, int month, int day)：指定日期

  • int getYear()：年

  • int getMonthValue()：月

  • int getDayOfMonth()：日

  • DayOfWeek getDayOfWeek()：星期几

    • getValue()：1～7，表示星期一～星期日

  • LocalDate plusDays(int n)：加 n 天

  • LocalDate minusDays(int n)：减 n 天
